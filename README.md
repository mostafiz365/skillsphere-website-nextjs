SkillSphere – Online Learning Platform

Project Purpose:
SkillSphere is a modern online learning platform where users can explore skill-based courses, view detailed lessons, and enroll in programs like Web Development, Design, and Marketing. The platform focuses on clean UI, responsiveness, and smooth user experience.

Live URL : https://skillsphere-website-nextjs.vercel.app

Key Features:

Authentication System:
Email & Password Login/Register
Google Social Login
Protected Routes (Course Details page)

Course Management:
Display all courses dynamically
Course Details page with full information
Top-rated courses shown in Popular section

Search Functionality:
Search courses by title in All Courses page

User Profile System:
View logged-in user profile
Update name & profile image


Technologies Used :
Framework: Next.js (App Router)
UI Library: Hero UI
Authentication: Better Auth
Styling: Tailwind CSS
Icons: React Icons
Animations: Framer Motion
Notifications: React Toastify
Data Handling: JSON / API Fetch
